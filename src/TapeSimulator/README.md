# TapeSimulator

Contributed by David Hilowitz <dhilowitz@gmail.com>

## Description

A vintage tape emulation effect featuring wow and flutter modulation, tape saturation, noise, and frequency filtering to recreate the character of analog tape machines.

The effect includes:
- **Wow**: Slow pitch modulation (1.2 Hz) with nested modulation for organic variation
- **Flutter**: Fast pitch modulation (6 Hz) for tape speed irregularities
- **Saturation**: Soft-clipping tape compression and harmonic distortion
- **Noise**: Tape hiss and surface noise
- **Filters**: Continuous lowpass/highpass tone shaping
- **Dry/Wet Mix**: Blend processed and dry signals

### Controls

| CONTROL | DESCRIPTION | NOTES |
|-|-|-|
| KNOB 1 | Wow Amount | Controls slow pitch modulation depth (±4% max) |
| KNOB 2 | Dry/Wet Mix | Blends between dry (left) and wet (right) signal |
| KNOB 3 | Filter | **LEFT** - Lowpass filter (300Hz-22kHz)<br/>**CENTER** - Filter off<br/>**RIGHT** - Highpass filter (20Hz-2000Hz) |
| KNOB 4 | Flutter Amount | Controls fast pitch modulation depth (±2% max) |
| KNOB 5 | Saturation | Tape saturation/compression (up to 10x drive) |
| KNOB 6 | Noise Amount | Tape hiss level |
| SWITCH 1 | Unused | **UP** - <br/>**MIDDLE** - <br/>**DOWN** -  |
| SWITCH 2 | Unused | **UP** - <br/>**MIDDLE** - <br/>**DOWN** -  |
| SWITCH 3 | Unused | **UP** - <br/>**MIDDLE** - <br/>**DOWN** -  |
| FOOTSWITCH 1 | Unused |  |
| FOOTSWITCH 2 | Bypass | The bypassed signal is buffered |
